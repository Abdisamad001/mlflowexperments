## 🚀 Setup Instructions

1. **📥 Clone the repository**

git clone https://github.com/Abdisamad001/mlflowexperiments.git
cd mlflowexperiments

### 📦 Install dependencies
pip install -r requirements.txt

### 📋 Requirements
- mlflow==2.5.0
- pandas==2.2.3
- numpy==1.26.4
- scikit-learn==1.5.2
- wget==3.2
- dagshub==0.3.42
- matplotlib==3.9.2
- scipy==1.13.1

### ▶️ Run the experiment
python app.py
